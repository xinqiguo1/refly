# 🚀 Refly 1.1.0 发布说明

一个专注于开发者效率、工作流透明度和智能体基础设施进阶治理的重磅更新。

Refly 正在从一个创作引擎进化为首个构建稳定、原子化且版本化的 Agent Skill（智能体技能）的开源平台。在 v1.1.0 中，我们将自然语言的力量引入了您的终端，并为您提供了对工作流执行与扩展的前所未有的控制力。

## 🌈 亮点

### 💻 CLI 进化：Skill 触手可及
通过自然语言在命令行中管理您的整个智能体基础设施。无需离开终端，即可创建、运行和分享 Skill。

### 🔍 工作流透明度提升
不再猜测智能体内部发生了什么。通过实时运行日志和即时产物预览，工作流的“黑盒”现在变成了“玻璃盒”。

### ⚙️ 进阶治理与运营
从自定义积分有效期到定时任务的多模型路由，Refly 现在提供了部署生产级 Agent Skill 所需的企业级控制能力。

## 🌟 新功能

### 💻 CLI 工具：开发者的利器
Refly CLI 经过全新设计，成为您开发 Skill 的主要接口：

- **自然语言 Skill 管理**：使用简单的自然语言命令即可创建、运行、导出和分享 Skill。
- **文件上传支持**：直接从 CLI 将本地文件无缝集成到远程工作流中。
- **复杂变量处理**：支持向工作流传递结构化数据和复杂参数，确保 100% 的可靠性。

### 🔍 工作流与执行控制
我们增加了多项功能，使构建和调试工作流变得更加快速和直观：

- **即时产物预览**：直接在工作流产物面板中预览文本文档和代码文件。
- **详细运行日志**：通过实时日志追踪任务执行的每个步骤，精确定位发生的情况。
- **细粒度节点控制**：支持单个节点独立运行，无需重新运行整个工作流即可进行快速测试和调试。
- **工作流管理**：支持随时修改工作流名称，并可查询特定节点的详情，保持工作空间井然有序。

### 🌐 生态与运营
Refly 继续扩展其连接能力与可靠性：

- **字节跳动生图工具**：新增原生支持的高质量图像生成工具。
- **灵活的定时调度**：定时任务现在支持按小时间隔运行，满足更频繁的自动化需求。
- **多模型路由**：为定时任务实现多模型随机或加权路由，优化成本或性能表现。
- **优化工具授权**：更顺畅的工具检查与授权流程，让集成变得前所未有的简单。
